# Stock Ranking Momentum Analyzer
Full version available here : https://pikoarare.gumroad.com/l/vhlxvg

Analyze stock ranking changes and detect momentum signals using ranking comparison.

This tool helps identify sudden market activity from ranking data.

Stock Ranking Analyzer is a Python-based toolset for analyzing stock ranking changes and momentum.

It helps identify stocks with sudden ranking movements using ranking comparison and momentum analysis.

## Features

• Collect stock ranking data 
• Compare rank changes between datasets 
• Detect momentum signals 
• Export analysis results to Excel or CSV
• Generate image of top 10 based on selected status from comparison results

---

## Tools Included

### 1️⃣ Ranking Data Collector

Collect stock ranking data from ranking pages and export to Excel or CSV.

Features

- Multiple ranking categories
- Automatic data formatting
- Excel / CSV export

---

### 2️⃣ Ranking Comparison Tool

Compare two ranking datasets and analyze changes.

Features

- Rank change detection
- Momentum calculation
- Status classification
- Excel output with highlighted signals
- generate simple images based on selected status from comparison results

---

## Example Output

Excel output includes:

• Rank change 
• Momentum score 
• Status signals (breakout, surge, drop, etc.)

Image output:

Select status to generate images of Top 10 based on status results.
You can generate simple images of top 10 based on selected status results
from comparison results for easy sharing and visualization.

Status signals of image

- BreakOut
- Out of Rank
- Surge
- Rebound
- Crash
- Hot
- Unchanged
